代数構造の構成

素朴な疑問

我々は気付いた時にはものを「数える」ことを知っている．やがて小学校の算数で，「数える」ことを使って「足し算」や「引き算」を学び，必至になって九九を憶え，それをもとに「掛け算」，さらに「割り算」を練習する．そのうち，「小数」を勉強して位取りを間違えたり，「分数」を勉強して通分を間違えたりする．色々な計算を練習するうちに，「交換法則」「結合法則」「分配法則」といった法則があるということを学び，それを使って計算の工夫をするよう教わる．

中学数学になると，今までいつでもできるわけではなかった「引き算」を自由にできるようにするために「負の数」が導入され，「整数」の概念が持ち込まれる．さらに，今まで「分数」と呼んでいたものがいつの間にか「有理数」と言われるようになり，その後すぐに「平方根」を始めとする「無理数」が出て来る．高校数学になると「有理数」と「無理数」を併せたものが「実数」であると教わり，高校数学も終わりごろになると「ベクトル」や「複素数」になって多くの学生を当惑させる……

ところで，散々云っているその「数」とは，いったいぜんたい何なのだろう？小学校で習ったように，数えられるものが「数」なのだろうか？しかし高校数学になって「実数」を学ぶと，もはやそれは指を折り曲げて数えられるものではない．辞書を引けば，同じ意味の「ものの順序を示す呼び名。序数。」とあるほか，「ものの数量や分量を示す呼び名。」ともある．これは恐らく，その背後に数量を「測定する」という物理的な意味合いも含まれているだろう．あるいは，“1” とか “1.4142” とか，こういった記号そのものが「数」なのだろうか？しかし $10$ 進法以外の位取り法を学べば，同じ記号を用いても別の値を示す場合があることを知る．「数」とは何であるかを明確に答えることができないまま，一方でその計算は一定のルールに従えば大抵実行できるのだ．こうして考えてみると「数」というものは，それこそ「命」のように，それが何であるか明確に述べることが非常に難しいが，それ自体はある程度感覚的に捉えられているのだ*1．

代数系

しかし，言葉や式をもちいて概念を厳密に定義し，そこから妥当な推論をして様々な事実の証明を行う，というのが数学の基本姿勢だから，やはり「数」とは何であるかもハッキリと定義付けたい．「数」はいまいち捉えどころのない概念だが計算は出来る，であればむしろ，その《計算》に注目する切り口で「数」を定義づけることにしよう，というのが現代流の「数」の定義である．ここで《計算》というのは，数に定義されている《演算》とそのルール，つまり《演算則》からなる体系のことである．この理念を，既に用意した集合や写像の概念を使って実行しよう．

《演算》とひとくちに云ってもその意味するところは結構広い．たとえば既に知っている「足し算」や「掛け算」のような演算は，いずれも「二つの数から一つの数を作る演算」である; 言い換えれば「二つの数にひとつの数を対応させる写像」である．このような演算を二項演算 (binary operation) と謂う．一般に，数の演算とは複数個の数にひとつの数を定める写像のことを謂う．「数の体系」というのは，集合の上に演算を定め，その演算に特定の演算則の成立を要請することで構成される —— つまり，「数」というのは自然の中にあるものではなくて，人間が規定するものなのだ：

空でない集合 $X$ (台集合 (underlying set) と呼ばれる) と，その上に定められた演算の組 $O$ とを併せた数学的対象を代数構造 (algebraic structure) と呼ぶ．
特に，集合 $X \neq \emptyset$ にひとつの二項演算を定めた代数構造を「マグマ」と謂う．これは「足し算」や「掛け算」が定められた数の体系の大本にある代数構造だ．このアツアツの「マグマ」の二項演算に様々な演算則を要請していくことで，徐々に具体的な数の体系が構成されていく，という流れだ．

集合 $X \neq \emptyset$ に二項演算 $* : X \times X \to X$ が定まっているとき，$(a,b) \in X$ に対して $*(x,y)$ を $x * y$ と書く．これは足し算のプラス記号などを意識した，二項演算特有の表記である．

群

集合 $X$ の上に二項演算 $*$ を定めたマグマ $(X,*)$ から，幾つかの演算則の成立を要請して代数構造を構成しよう．小学校の算数では交換法則とか結合法則といった演算「法則」は，何度もいろいろな計算を行って確かめたうえで認めたものだった (だから法則と呼ぶのである)．しかし今の立場では，こういった規則が成り立つことを二項演算に要請するのだ．その気分を表すときには「演算則」と，また「交換則」「結合則」「分配則」，あるいは「交換律」「結合律」「分配律」と呼ぶ．演算則自体はいろいろなものを考えることができるが，その中でもとりわけ重要なのが次の三つだ：

  1. (G1) $\forall a,b,c \in X \big[ (a * b) * c = a * (b * c) \big]$．
  2. (G2) $\exists e \in X \big[ \forall a \in X( e * x = x * e = x) \big]$．
  3. (G3) $\forall a \in X \big[ \exists b \in X (a * b = b * a = e) \big]$．
これら三つの演算則を満たすような二項演算 $*$ を備えたマグマは群 (group) と呼ばれる．その構造の単純さゆえに非常に幅広い応用を持つ基本的な代数構造である．

1 番目の条件 (G1) は結合律 (associative property) と謂う．これは当たり前のことのように思われるかも知れないが，あくまでもこの性質の成立は人間が決めるものなのだ．実際，結合律を満たさないような数の体系も存在する．しかし結合律が要請されていれば，演算をとる順序は自由に入れ替えても良いから，$(a * b) * c$ も $a * (b * c)$ も特に気にせずに単に $a * b * c$ と書くことが許されるのだ．

二番目の演算則 (G2) で存在を要請された元 $e$ を単位元 (identity) と謂う．単位元は存在するならば唯一つである：もし $e'$ もまた群の単位元なら，単位元の定義から $e' = e'e = ee' = e$ であるから．また，三番目の演算則 (G3) により任意の元 $a \in X$ に対して存在を要請された元 $b$ を，$a$ の逆元 (inverse) と謂い，記号 $a^{-1}$ で表される．これも存在すれば一意である：$c \in X$ も $a$ の逆元なら，定義より \[ c = c * e = c * (a * a^{-1}) = (c * a) * a^{-1} = e * a^{-1} = a^{-1} \] であるから．逆元の存在が要請されているとき，任意の $a,b \in X$ に対して $a * c = b * c$ なら，両辺に右から $c^{-1}$ をかけて $a = b$ を得る．同様に，$c * a = c * b$ なら両辺に左から $c^{-1}$ をかけて $a = b$ を得る．こうして，ある種の「割り算」のようなことが出来るようになっている．

以上の三つの演算則に加えて次の演算則も成り立つとき，群 $(X,*)$ は可換群あるいはアーベル群 (Abelian group) と謂う：

  1. (G4) $\forall a,b \in X \big[ a * b = b * a \big]$．
これは可換律 (commutative property) ，つまり交換法則に相当するものだ．例えば，我々がよく知っている普通の意味での「足し算」，恰好良く言えば加法 (addition) によって，「整数」全体，「有理数」全体，「実数」全体はいずれもアーベル群になっている*2．この「足し算」の類推で，アーベル群の群演算はしばしば記号 $+$ で表される．このとき，加法に関する $a\in X$ の逆元は普通 $-a$ と表されて，$a,b \in X$ に対し $a +(-b)$ をとる演算を「引き算」，恰好良く言えば減算 (subtraction) と呼んで記号 $a -b$ で表す．

マグマに結合律と単位元の存在だけを要請して，逆元の存在を要請しない場合はモノイド (monoid) と呼ばれる代数構造になる．これも群と同様，代数学においてしばしば現れる基本的な代数構造である．

環・体

代数構造に定義される演算は複数あっても良い．例えば，既に知っている「実数」は「足し算」と「掛け算」の二つの演算が組み込まれている．このような代数構造の土台になるのが《環(かん)》である：

$(R,+)$ をアーベル群とする．これにもう一つの二項演算 $\cdot : R \times R \to R$ が定められ，$(R,\cdot)$ はモノイドであるとする．つまり二項演算 $\cdot$ について結合律が成り立ち，$\cdot$ についての単位元が存在する．これに加えて条件

  • (R1) $o \neq e$，$o$ は加法 $+$ の単位元，$e$ は演算 $\cdot$ の単位元．
  • (R2) $\forall x,y,z \in X \big[ x\cdot (y +z) = x\cdot y +x\cdot z \big]$．
  • (R3) $\forall x,y,z \in X \big[ (x +y)\cdot z = x\cdot z +y\cdot z \big]$．
が成り立つとき，代数構造 $(R,+,\cdot)$ を環 (ring) と謂う．条件 (R2)，(R3) はそれぞれ左分配律，右分配律と謂い，これらを併せて分配律 (distributive property) と謂う．環の二つ目の二項演算 $\cdot$ は普通乗法 (multiplication) と呼ばれ，しばしば $x\cdot y$ を略して単に $xy$ と表す．環の乗法が可換律 $xy = yx$ も満たすとき，$(R,+,\cdot)$ は可換環 (commutative ring) と謂う．

環の加法単位元 $o$ は次の特別な性質を持っている：任意の $x\in X$ に対し， \[ ox = (o +o)x = ox +ox, \qquad\therefore\quad ox = o. \] 同様に $xo = o$．このような性質を持つ元を零元 (zero) と謂い，整数や実数の $0$ との類推で，しばしば記号 $0$ で表される．普通は環の加法単位元を零元の名前で呼び，環の単位元と謂えば乗法についての単位元を意味する．こちらは整数や実数の $1$ との類推で，しばしば記号 $1$ で表される．たとえば「整数」全体は既に知っている意味での加法と乗法に関して環を成している．

環 $(R,+,\cdot)$ について，その台集合 $R$ から零元 $o$ を除いた集合 $R\setminus\{o\}$ が乗法に関してアーベル群となるとき，つまり零元以外の全ての元に対して乗法逆元が存在するような可換環 $(R,+,\cdot)$ を体 (field) と謂う．「有理数」全体，「実数」全体，「複素数」全体はいずれも体の典型例である．

環および体について以下が成り立つ：

  1. $1(-1) = (-1)1 = -1$．
  2. $a(-1) = (-1)a = -a$．
  3. $a(-b) = -ab = (-a)b$．
  4. $(-1)(-1) = 1$．
  5. $(-a)^{-1} = -(a^{-1})$．
  6. $(ab)^{-1} = a^{-1}b^{-1}$．
これらはいずれも分配律を用いれば容易に示される．
